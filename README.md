## Getting Started

You can work on this project using any preferred workflow: local development, cloud IDEs, or direct repository editing.

### Prerequisites

Ensure you have the following installed:

- Node.js (latest LTS recommended)
- npm or yarn
- Git

> Recommended: Install Node.js using [nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

---

## Local Development Setup

Follow these steps to run the project locally:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start development server
npm run dev
```

The application will start with hot-reloading enabled and can be previewed in your browser.
